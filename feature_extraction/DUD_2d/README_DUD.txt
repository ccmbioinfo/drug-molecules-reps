'DUD_training_list' contains the indices of the compounds used for training for each target using five query molecules.

'DUD' contains the compound list of each target with [external ID, internal ID and SMILES]. A total of 21 targets.

This database is downloaded from reference [1].

[1] Riniker, Sereina, and Gregory A. Landrum. "Open-source platform to benchmark fingerprints for ligand-based virtual screening." Journal of cheminformatics 5.1 (2013): 1-17.
